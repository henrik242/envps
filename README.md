procenv
-------
Show process environment variables
```
$ procenv 242
SERVICE_NAME=example
PATH=/usr/bin:/bin:/usr/sbin:/sbin
```

Installation
------------
Just run `make` or install with Homebrew/Linuxbrew:
```
$ brew install henrik242/brew/procenv
```
